# aap-juce-aelapse

It is a port of [smiarx/aelapse](https://github.com/smiarx/aelapse) to [Audio Plugins For Android (AAP)](https://github.com/atsushieno/aap-core), using [aap-juce](https://github.com/atsushieno/aap-juce).

The app icon is hackily created from the screenshot.
